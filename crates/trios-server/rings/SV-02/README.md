# SV-02 — A2A Agent-to-Agent Protocol

## Purpose
Agent registration, discovery, and task assignment via A2A endpoints.

## Endpoints
- `a2a/register` — register agent with AgentCard
- `a2a/list_agents` — list all registered agents
- `a2a/send` — send message between agents
- `a2a/assign_task` — assign task to agent
