\chapter{Coq Citation Map}
\label{app:coq-map}

This appendix maps every theorem cited in the monograph to its Coq
mechanization. The single source of truth is
\href{https://github.com/gHashTag/trios/blob/main/assertions/igla_assertions.json}{\texttt{assertions/igla\_assertions.json}}.
Status is reported honestly: a theorem is \texttt{Proven} only when
the corresponding \verb|.v| file ends with \verb|Qed.| and the
governing \verb|Admitted|-count gate is zero.

\section*{Honesty audit}

The audit utility \verb|cargo run -p trios-phd -- audit --coq-map|
verifies that:

\begin{itemize}
  \item every \verb|Admitted| in the table corresponds to an actual
        \verb|Admitted.| line in the cited \verb|.v| file (no false
        \texttt{Proven});
  \item every theorem cited by name in the monograph appears here;
  \item the count of \verb|Admitted| entries does not exceed the
        contractual ceiling defined in
        \verb|assertions/igla_assertions.json|.
\end{itemize}

A non-zero exit from this audit is a build-blocking R5 violation.
